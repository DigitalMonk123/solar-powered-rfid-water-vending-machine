# solar-powered-rfid-water-vending-machine
A smart IoT-based water vending system built using ESP32, RFID authentication, and solar-powered embedded automation for intelligent water dispensing applications.

## Features
• RFID-based access control
• Solar-powered operation
• Automatic water dispensing
• ESP32 embedded control
• Low-power system design
• Relay-controlled pump automation
• IoT-ready architecture
• Smart vending functionality

• Embedded C/C++

• ESP32 Firmware
• SPI Communication

## Working Process
• User scans RFID card
• ESP32 verifies authentication
• Relay activates water pump
• Water dispenses automatically
• System stops after fixed timing
